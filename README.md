=========================
Home page
=========================

Shows all the existing users and the other user is able to view details of a specific user's profile, such as, their inventory and their total points. 

=========================
Task page
=========================

Shows all the tasks the user can embark on to earn points, a success page will be shown and points will be added to their profile  after the user has completed their task after clicking on "Embark".

If the user has not logged in, the "Embark" button will be removed, and the header will ask the user to log in or register in order to embark the Task.
=========================
Quest page
=========================

Shows all the quests the user can embark on to get a random reward item, a success page will be shown and the reward will be added to their inventory after the user has completed their quest after clicking on "Embark". The quests are of a higher difficulty than the tasks.

If the user has not logged in, the "Embark" button will be removed, and the header will ask the user to log in or register in order to embark the Quest.
=========================
Creature Shop page
=========================

Shows all the creatures that the user can buy using their points, after clicking on the "Purchase" button, a success page will be shown if the user has enough points to purchase the creature, and a insufficent point page will be shown if the user does not have enough points to purchase the creature. After purchasing, points will be deducted from the user's total points and the creature will be stored inside the inventory. 

If the user has not logged in, the "Purchase" button will be removed, and the header will ask the user to log in or register in order to purchase the creature.

=========================
Item Shop page
=========================

Shows all the item that the user can buy using their points, after clicking on the "Purchase" button, a success page will be shown if the user has enough points to purchase the item, and a insufficent point page will be shown if the user does not have enough points to purchase the item. After purchasing, points will be deducted from the user's total points and the item will be stored inside the inventory. 

If the user has not logged in, the "Purchase" button will be removed, and the header will ask the user to log in or register in order to purchase the item.

=========================
Message page
=========================

It is global page where users can write global messages, and edit or delete their own message. Each user can view all messages written by users.

If the user has not logged in, the message bar will be removed and they cannot edit or delete their own message, and the header will ask the user to log in or register in order to do so.

=========================
Profile page
=========================

It shows the user's own information and their own inventory. The user can also feed and water their creature in their inventory.

If the user has not logged in or registered an account, the navbar will now show the Profile page.

=========================
Logout page
=========================

Allows the user to log out of their account.

=========================
Login page
=========================

It shows a form where the user have to enter their username and password in order to log into their own account.

=========================
Register page
=========================

It shows a form where the user have to enter their username, email, password and confirm password in order to register into a new account.
